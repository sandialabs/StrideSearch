TropicalStormListNode

Extends the StormListNode type for tropical cyclone detections. More...

Detailed Description

Extends the StormListNode type for tropical cyclone detections.

Like its parent StormListNode, TropicalStormListNode procedures are divided into 2 categories: Node procedures that work on one or two nodes directly, and list procedures that work with node pointers.
